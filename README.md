## Why Sapling exists

Most productivity tools push urgency, metrics, and constant optimization.
Sapling takes a different approach.

Sapling believes:

- Rest is part of growth, not a failure
- Progress compounds quietly
- Missing days should never erase effort

Sapling is designed for students, creatives, and anyone tired of being rushed by their tools.
